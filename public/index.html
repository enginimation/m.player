How to use

m.player - is modern offline audio player that works just in browser. You can drop music files from your file sytem inside the player and play them right now or come back later.

Player has library using which you can navigate through you music. Library populated automatically each time you add new files to play. Information in the library integrated with last.fm service.

All you music stored locally in the filesystem used by application. This is possible using HTML 5 technology.

Project is open-source. You can join development here.

Hot keys

Space
Play/Pause
Enter
Play selected track
Key up
Select previous track
Key down
Select next track
